# rags-drum-essay

一个 **Claude / Claude Code 技能（skill）**：为「屋顶现视研·拾荒战略 Rags Drum」年度 ACG 征文——含 2026 年复活的**「拾荒战略 Repeat Vol.1・重思日常」**——构思、写作或修改参赛文章时，载入一份从 **23 篇历届获奖作品**蒸馏出的评审偏好画像：统计事实、一手赛制规则（2020–2022 三届完整公告 + Repeat Vol.1 新规）、分级置信度的偏好推论，以及选题/结构/文体对齐清单与终稿自查。

> 它不是模板，而是**先验分布**。机械套用获奖模板本身可能被该社群识别为套路（社群奖励过对"套理论"风气的自我批判）。正确用法是拿它去**校准判断**，而非填空。

## 认识论前提（重要）

- 语料 = 23 篇**获奖**作品（2020×10、2021×8、2022×5，与三届官方受赏名单交叉核实），约 26 万字。这是**幸存者样本**：没有落选稿基线（落选侧信息仅有 2021 公告的 3 篇落选题目与 1 条官方理由，外加各届前/后夜祭层级差）。
- 2026 年的「拾荒战略 Repeat Vol.1」已改为**单轮评选**（无前夜祭/后夜祭、无提要通道）、主题限定「重思日常」，并**新设生成式 AI 使用条款**——本技能定位是资料与自查清单（校准判断的先验），按官方条款，AI 只能用于润色、文献检索、数据整理等非核心环节且须声明；架构与论点必须出自作者本人。
- 文件区分三级断言：**【事实】**＝对文本的直接测量或公告原文；**【推论·强/中/弱】**＝对"评审偏好"的后验猜测。任何"评审喜欢 X"都是从中奖分布反推的先验，不是评分规则。
- N=23 且跨三届、评委逐届可能更换，**谨防过拟合**。
