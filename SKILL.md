# 拾荒战略参赛写作对齐指南

## 0. 认识论前提（先读，不可跳过）

- 语料 = 23 篇**获奖**作品（2020×10、2021×8、约2022×4、届次不明×1），约 26 万字。这是幸存者样本：**没有落选稿基线**（仅 2021 公告透露 3 篇落选标题与 1 条官方落选原因）。
- 本文件区分三级断言：【事实】= 对文本的直接测量或公告原文；【推论·强/中/弱】= 对"评审偏好"的后验猜测。任何"评审喜欢 X"都是从中奖分布反推的先验，不是评分规则；评委逐届可能更换，N=23 且跨三届，**谨防过拟合**。
- 反身警告【强推论】：该社群奖励过对自身"套理论"风气的批判（凉宫文正以此获奖），也有获奖文自带编辑质疑夹批。机械模仿获奖模板本身可能被识别为套路。对齐的正确用法是：把下文当先验分布去校准判断，而不是当模板去填空。

## 1. 赛制事实【一手：2021 评选公告，详见 references/contest-facts.md】

- 两级评审：**前夜祭** = 文章完整性 + 评委多数通过；**后夜祭** = 评委一致通过，可空缺（2021 年后夜祭零篇通过）。通过前夜祭即有 300 元稿酬。
- 存在"先交提要、后交正文"通道；官方脚注点名过一篇落选稿：**正文未充分兑现提要主张，影响了投票**。→ 提要是承诺书。
- 2021 评委 5 人：will、马小褂、有电拍拍、rocefactor、屋顶第十一任所长（其他届未知，不可外推）。
- 形式开放：文章/视频论文/录音；语言中/英/日皆可（语料中全日语论文、视频讲稿版均获过奖）。
- 公告以布莱希特题词开篇；2020 公告引卡夫卡论。社群自我定位带批判理论底色。

## 2. 获奖文本统计画像【事实，详见 references/corpus-stats.md】

- **字数**：中位 10454，主体区间 7k–15k；极值 5006 与 23797 都获过奖。
- **结构**：分节几乎普遍（19/23 用小标题，余者用加粗行/目次）；小标题以"数字+短语"最多；4 篇带摘要/提要+关键词。
- **标题**：约 18–28 字；16/23 为「主：副」或「——」双段式，多含自造概念或悬念设问。
- **配图**：23/23 有图，中位 6 张（知乎成文形态）。
- **注释**：9/23 有文末参考文献区块；从零注释到 49 条尾注都获过奖 → 引注规范是可选项。
- **互文**：每篇提及作品中位约 14 部，最高 62 部。
- **理论家出现篇数**：齐泽克 5、东浩纪 5、拉康 4、福柯 4、黑格尔/阿甘本/大塚英志/鲍德里亚/巴赞/本雅明/德勒兹各 3；另有莲实重彦、宇野常宽、大澤真幸、梅洛-庞蒂、唐·伊德等重仓篇目。高频概念：他者(10)、主体性(8)、意识形态(7)、萌(6)、后现代(6)。
- **题材**：动画 7、游戏 5、文学/轻小说 3、文化史怪物 2、跨媒介作品群 1、漫画 1、类型社会诊断 1、媒介理论 1、网络亚文化概念 1、日语文明论 1。
- **负空间**：语料中不存在安利文、评分文、资讯文、榜单文、纯情绪杂谈；主对象无国创作品。
- **人称**：集体"我们"为主流；论战文与分析哲学文用强"我"；量化论文用"本文"。

## 3. 评审偏好模型【推论，按置信度分层】

**强**（语料内近乎无例外，且六个独立精读组结论收敛）：
- P1 主评价轴 = **理论纵深 × 与文本的咬合度**。通行证是"低对象 × 高理论"的反差：眉毛透不透→梅洛-庞蒂；网络梗"云玩家"→分析哲学充要条件；赌博漫画→鲍德里亚。评的不是"谈了什么二次元"，而是"用多硬的刀法切它"。
- P2 **自造概念/命名是辨识度资本**：约八成获奖篇有可记忆的中心命名或二元对（"停滞的风格""纯爱矩阵""空间化的身体VS身体化的空间""惧兽""近代の非超克"）。结尾把对象钉进批评史坐标的"命名式加冕"是高频获奖动作。
- P3 **问题驱动开场**："何以/是哪种意义上的"式反常识设问、题记或理论断言进入，最好在开头把核心问题形式化为 1–2 个明确问句；不做剧情综述。
- P4 **收束不封闭**：升华宣言/开放诘问/机锋反转 + 回扣标题；平铺的"总结陈词"几乎不存在。
- P5 理论要与**桥段/机制/分镜级**的具体文本证据逐点咬合；理论与例证平行陈列（各说各话）是获奖作中被批评的下限状态，不是卖点。
- P6 **完整性与提要兑现**是仅有的两条接近官方明示的标准（前夜祭规则 + 落选点名）。

**中**：
- P7 理论配置常态 = **欧陆轴 + 日本批评轴**双线（拉康/齐泽克/福柯/德勒兹/阿甘本/鲍德里亚/本雅明 × 东浩纪/大塚英志/宇野常宽/莲实重彦/大澤真幸）；东浩纪谱系接近社群"母语"，即便反驳他也要先接住他。
- P8 **元批评自觉加分**：用理论的同时对"套理论"保留一处反身警惕。
- P9 **方法稀缺性本身是资本**：量化爬虫+NLP、图论形式化、分析哲学、全日语学位论文体、显微逐镜头细读——每种"全场唯一"的方法都获奖了。敢用没人用过的正经方法。
- P10 **文字打磨是弱约束**：错别字、病句、臃肿、自称"半成品"都被容忍过；力气应优先押在框架、概念与细读，而非文句精修（但别把这读成"可以烂尾"——完整性是硬标准）。
- P11 政治/批判锋芒不减分：国际歌收尾的文章全票通过 2020 后夜祭；左翼批评姿态与社群自我表述同频。

**弱**（N 小、届次混杂，仅供参考）：
- P12 后夜祭一致通过与字数、注释规范无可见相关；"写长写全"不是策略。
- P13 负样本仅 3 篇标题：单季介绍向、"后学考察"式套路题也会在后夜祭被否——熟悉的套路无保送。

## 4. 写作对齐操作

**A. 选题构思**：公式 = 具体 ACG 对象 ×"何以"问题 × 预留一个可自铸的概念名。已验证的六型：
1) 单作深读（大理论解一部冷门/被低估作品）；2) 作品群策展（一条理论线索串多部作品编年）；3) 类型症候诊断（从流行套路读社会情绪）；4) 媒介/技法本体论（把作画、演出、界面等技术细节理论化）；5) 量化实证（爬虫/统计检验圈内直觉）；6) 概念分析（给圈内梗做哲学解剖）。
检验问句：能否一句话回答"本文命名了什么？"

**B. 动笔**：
- 结构模板：题记（可选）→ 设问/理论开场（≤3 段内亮出问题）→ 3–7 节"数字+短语"小标题，每节一个论证环节 → 桥段级细读验证 → 结尾回扣标题/开头，命名式论断 + 升华或开放诘问。
- 若走提要通道：先写 200–400 字提要当承诺书，正文严格兑现其主张。
- 文体：学术-随笔混合；长句思辨 + 格言短句独立成段制造顿挫；关键术语括注外文原词（セカイ系、Divine Violence、écriture）；关键词适度加粗；引原文用引用块。人称按文类选：引导式"我们"（默认）/强"我"（论战、分析哲学）/"本文"（论文体）。
- 参数：字数瞄 8k–15k；段落 100–250 字；配图 4–10 张（截图/分镜/图表）；文末参考文献建议给（非门槛，但零成本的专业信号）。

**C. 终稿自查**（模拟两级评审）：
- [ ] 核心问题在开头 500 字内被形式化？
- [ ] 有一个属于本文的自造概念/命名，且结尾完成"命名式加冕"？
- [ ] 每个理论环节都有具体文本/画面/机制证据咬合，而非平行陈列？
- [ ] 欧陆轴 + 日本批评轴至少各一，术语附原文？
- [ ] 互文视野 ≥10 部相关作品？
- [ ] 提要与正文主张一致？
- [ ] 结尾回扣标题，升华或开放诘问？
- [ ] 对"套理论"有至少一处反身自觉？
- [ ] 完整性：论证闭环、无断头节、无"下篇再续"？
- [ ] 一致通过压力测试：假想 5 位挑剔评委各自最可能否决本文的一条理由，并逐条处理或自觉接受。
