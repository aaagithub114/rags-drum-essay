# 获奖样本对齐指南

仅在用户需要对标历届获奖样本、诊断结构与文体或执行终稿自查时读取本文件。赛制与官方要求见 `contest-facts.md`，定量依据见 `corpus-stats.md`，具体作品档案见 `work-profiles.md`。

## 使用边界

- 将下列内容视为从 23 篇获奖作品归纳出的先验，不视为官方评分规则。
- 保留【强/中/弱】置信度。没有完整落选稿基线，不能由获奖样本直接推出因果关系。
- 根据对象、文类和作者论证选择建议；不要机械填充理论家、概念、配图或互文数量。
- 本文件只包含原创分析与必要的短例，不包含或替代获奖作品原文。

## 评审偏好模型

### 强推论

- **理论纵深与文本咬合度共同构成主要辨识度**：让理论解释具体桥段、机制、分镜或媒介形式，不要让理论与例证平行陈列。
- **自造概念或中心命名经常提高辨识度**：仅在它确实压缩并解释全文论证时使用，不要为了形式而造词。
- **问题驱动开场是高频实践**：尽早明确“何以”或“在哪种意义上”的核心问题，避免用大段剧情综述代替问题。
- **结尾通常回扣开头与标题**：完成论证后保留开放诘问、反转或进一步问题，避免只复述章节摘要。
- **完整性和承诺兑现最接近硬标准**：保证标题、摘要或导语提出的主张在正文得到充分处理。

### 中推论

- 欧陆理论与日本批评经常并用，但这不是每篇文章都必须满足的配置。
- 对“套理论”保持反身警惕，通常比无条件展示理论更符合样本气质。
- 稀缺且可靠的方法可能形成辨识度，例如量化分析、形式化论证、逐镜细读或媒介考据。
- 样本对文字瑕疵的容忍度似乎高于对框架、概念和细读缺失的容忍度；这不意味着可以忽略文字质量或完整性。
- 批判性或政治锋芒在样本中未显示为负面因素，但单个案例不足以推出稳定偏好。

### 弱推论

- 后夜祭结果与篇幅、注释规范未显示清晰关系；不要把“写长写全”当作策略。
- 少量落选信息显示，介绍向写法或熟悉的理论套路没有保送效果；由于负样本极少，只能作为风险提示。

## 选题构思

从“具体 ACG 对象 × 可回答的问题 × 可验证的方法”建立选题。只有在论证需要时，再考虑一个能够概括发现的中心命名。

可参考以下文类，不必受其限制：

1. 单作深读：用明确问题解释一部作品。
2. 作品群策展：用同一线索比较一组作品的变化。
3. 类型症候诊断：从流行套路分析社会情绪或文化机制。
4. 媒介或技法研究：分析作画、演出、界面、声音等形式机制。
5. 量化实证：用可复核的数据检验圈内直觉。
6. 概念分析：澄清圈内概念的边界、条件与反例。

处理 Repeat Vol.1 时，先确认选题如何回应“重思日常”，再从 `contest-facts.md` 第三、七节选择必要的官方语境；不要因为官方引言点名某位理论家就强行引用。

## 结构与文体

- 在开头建立问题、对象和论证承诺；不要把固定字数或段数当成硬要求。
- 让每一节承担一个清晰的论证环节，并用文本、画面、机制、史料或数据验证。
- 依据文章类型选择人称、术语密度和引注形式，不要模仿统计上的多数形式。
- 字数、标题、配图、段落和引注分布见 `corpus-stats.md`；仅在用户需要这些参数时读取。
- 需要具体结构或文体参照时，从 `work-profiles.md` 选择最相近的 2–3 篇，不要通读或仿写全部档案。

## 终稿自查

- [ ] 当前届次的主题、篇幅、投稿形式和 AI 条款是否满足？
- [ ] 核心问题是否在开头清楚出现？
- [ ] 标题、摘要或导语许下的论证承诺是否得到兑现？
- [ ] 每个关键理论环节是否有具体证据，而非理论与例证平行陈列？
- [ ] 中心概念是否真正解释材料，而非为了显得新颖而命名？
- [ ] 章节之间是否形成推进关系，且没有断头节或留待“下篇”的关键缺口？
- [ ] 结尾是否回答核心问题并回扣开头，同时保留必要的限度意识？
- [ ] 是否把样本倾向误写成官方标准，或为了对标而强加无关理论与形式？
- [ ] 能否列出最可能导致评委否决的三项理由，并逐项处理或明确接受风险？
